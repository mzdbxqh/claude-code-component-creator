# 通用报告渲染逻辑

## 模板填充引擎

**占位符语法**: `{{variable}}`

**步骤**:
1. 解析模板占位符
2. 替换为实际数据
3. 处理条件渲染 (if/else)
4. 处理循环渲染 (for)

## 后处理优化

1. 清理多余空行
2. 修正 Markdown 格式
3. 添加目录 (可选)
4. 检查链接有效性

## 错误处理

| 错误 | 处理策略 |
|------|---------|
| JSON 解析失败 | 返回详细错误信息 |
| 模板缺失 | 使用通用模板继续 |
| 占位符未匹配 | 保留原样并记录警告 |
| 文件写入失败 | 重试 1 次并报告 |
