# CCC Skills

This directory is reserved for user-invocable skills that extend the Claude Code Component Creator plugin.

## Purpose

Skills in this directory can be invoked by users through the `/ccc:skill-name` command pattern.

## Adding Skills

To add a new skill:
1. Create a subdirectory with the skill name
2. Add a `SKILL.md` file following CCC standards
3. Register the skill in the plugin configuration

## Current Status

This directory is currently empty. Skills are defined in the `commands/` directory and managed through the plugin system.
